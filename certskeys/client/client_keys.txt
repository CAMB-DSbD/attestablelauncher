To run the attestable launcher.py, place in the the
server and client directories the keys and certificates
expected in the RESOURCE_DIRECTORY:


client:
aliceClient.intermediate.chain.pem
aliceClient.key.pem
rootca.cert.pem 

These keys and certs can be generated by python or 
openssl. I used the latter. See
https://github.com/CAMB-DSbD/FEWD/blob/main/docs/howtocreateSelfSignedCerts.txt
